This file lists all the Python packages required to run the Tic-Tac-Crash game.
To install all these packages, run the following command in your terminal:
pip install -r requirements.txt
opencv-python
mediapipe
firebase-admin

A specific version of protobuf is needed to ensure compatibility
between mediapipe and firebase-admin.
protobuf==4.25.3
